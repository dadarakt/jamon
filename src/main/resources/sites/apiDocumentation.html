Search for functions by their name

getImplementationForMethod/searchTerm

Searches for the given term in function names only. Per default returns a list of all function names which include the search term. These names can then be used to retrieve methods for a explicit function using the 'findMethodsForFunction' resource.

Find Methods for a function

findMethodsForFunction/-functionName-

Given the fully qualified name of the function it will return a list of Julia method signatures which exist for the given function. The results can then be used to retrieve a particular implementation with source code for a particular method using the 'getImplementationForMethod' resource.

Get an implementation of a method

getImplementationForMethod/-methodName-

Allows for parameterized retrieval of source code of a method. Since there might be several implementations of the same method the user can decide if he wants to choose the most popular, the most recent, the fastest or most memory-efficient implementation.
